# 스페인어 관사 짝맞추기 게임 (Juego de Artículos en Español)


스페인어 학습자를 위한 인터랙티브 웹 기반 단어 학습 게임입니다. 헷갈리기 쉬운 스페인어 명사의 성별(남성/여성)과 그에 맞는 정관사(el, la)를 직관적으로 짝맞추며 재미있게 암기할 수 있도록 설계되었습니다. 

본 프로젝트는 2026년 SW프로그래밍의기초(SW PROGRAMMING BASICS)-07분반의 1학기 기말 과제로, AI를 활용한 **바이브 코딩(Vibe Coding)** 방식을 통해 기획 및 개발되었습니다.

---

## 🎯 기획 의도 및 학습 목표

스페인어를 배울 때 가장 큰 난관 중 하나는 명사의 성별을 구분하는 것입니다. `-a`로 끝나지만 남성 명사이거나, `-o`로 끝나지만 여성 명사인 '불규칙 명사'들은 단순 암기만으로는 한계가 있습니다. 

이러한 학습의 어려움을 해결하고자, 다음과 같은 목표를 두고 게임을 개발했습니다.
* **상호작용을 통한 암기 강화:** 시각적인 피드백(색상, 애니메이션)을 통해 정답과 오답을 즉각적으로 인지합니다.
* **실용적인 어휘 풀(Pool):** 뻔한 단어가 아닌, **실제 학습 중인 교재의 단어-CURSO DE ESPAÑOL 1: Inicial**들과 자주 틀리는 **불규칙 함정 단어**들을 중점적으로 배치했습니다.
* **의미 학습 병행:** 스페인어 단어와 한국어 뜻이 함께 나타나 관사와 의미를 동시에 학습할 수 있습니다.

## ✨ 주요 기능 (Features)

1. **클릭 앤 매치 (Click & Match)**
   * 무작위로 배치된 스페인어 명사 카드를 선택하고, 알맞은 관사(el, la 등) 버튼을 클릭하여 짝을 맞춥니다.
2. **동적 애니메이션 피드백**
   * **정답 시:** 카드가 초록색으로 빛나고 부드럽게 사라집니다. (Fade-out)
   * **오답 시:** 카드가 빨간색으로 변하며 좌우로 흔들리는 애니메이션(Shake)이 발생해 확실한 시각적 경고를 줍니다.
3. **랜덤 출제 시스템**
   * 전체 단어 데이터베이스에서 게임을 시작할 때마다 무작위로 10개의 단어를 추출하여 반복 학습의 지루함을 줄였습니다.
4. **게임 클리어 및 리셋**
   * 모든 카드의 짝을 맞추면 축하 메시지와 함께 '다시 하기' 버튼이 활성화되어 게임을 즉시 초기화할 수 있습니다.

## 🛠️ 사용 기술 (Tech Stack)

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Development Method:** Vibe Coding (Powered by Google Antigravity)
* **Deployment:** GitHub Pages

## 📚 주요 수록 단어 예시

* **불규칙 명사:** `el problema` (-a로 끝나는 남성 명사), `la mano` (-o로 끝나는 여성 명사), `la radio` 등
* **실제 학습 중인 교재의 단어:** `la arquitectura` (건축), `el escenario` (무대), `la exposición` (전시회) 등 총 40+ 단어 수록

## 🚀 실행 방법

웹 브라우저만 있다면 누구나 설치 없이 바로 플레이할 수 있습니다.
1. 상단의 링크(GitHub Pages)를 클릭하여 게임에 접속합니다.
2. 화면 오른쪽에 흩어진 명사 단어 중 하나를 클릭해 하이라이트 시킵니다.
3. 화면 왼쪽에서 해당 단어에 맞는 관사를 클릭합니다.
